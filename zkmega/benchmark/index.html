Benchmark

We have constructed the tests of zkMega curves in pallet-template which imports the curves from zkMega directly(Wasm) and with runtime-interface(Native).

Building#

# Clone the branch `curve-benchmark` of our forkgit clone https://github.com/patractlabs/jupiter.git \    --branch features/runtime-interfaces \    --depth =1
# Build the templatecargo build -p jupiter-dev --all-features --release
# Check the command benchmark works fine# ./target/release/jupiter-dev benchmark -p pallet_template -e wasm_bls_12_381_add./target/release/jupiter-dev benchmark -p pallet_template -e wasm_bls_12_381_add
