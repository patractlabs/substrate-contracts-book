    # 插件

    # 背景信息

    Redspot的核心功能包括Redspot runtime environment、编译合约、运行测试和运行脚本。其他功能，例如与合约的测试套件、合约交互、访问合约、设置不同的链的配置等，都由插件提供，您可以按需选择。引入一个插件的方式很简单，将它作为一个npm的依赖添加，然后在redspot.config.ts中加入即可。示例如下。

    import { RedspotUserConfig } from 'redspot/types';
import '@redspot/patract'; // 引入 @redspot/patract 插件
import '@redspot/chai'; // 引入 @redspot/chai 插件
export default {
 ...
} as RedspotUserConfig;

    现在提供的有以下这些插件：

    • @redspot/chai (opens new window)
    • @redspot/decimals (opens new window)
    • @redspot/explorer (opens new window)
    • @redspot/gas-reporter (opens new window)
    • @redspot/known-types (opens new window)
    • @redspot/patract (opens new window)
